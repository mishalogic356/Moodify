# 🌈 Moodify

✨ An interactive web experience that delivers personalized songs and inspirational quotes based on the user's mood, enhanced with immersive visuals and a responsive user interface.

## 🚀 About The Project

Moodify is a frontend web application that creates a personalized experience by providing mood-based music recommendations and motivational quotes.

Whether users are feeling happy, sad, relaxed, or motivated, Moodify presents suitable content along with visually engaging backgrounds to enhance their experience.

## ✨ Features

- 🎵 Mood-based song recommendations
- 💭 Inspirational and motivational quotes
- 🎨 Dynamic backgrounds for different moods
- 📱 Responsive design for all devices
- ⚡ Smooth and interactive user interface

## 🛠️ Built With

- HTML5
- CSS3
- JavaScript

## 🎯 Project Goal

The objective of Moodify is to combine music, motivation, and design into a single platform that helps users connect with their emotions through an engaging digital experience.
